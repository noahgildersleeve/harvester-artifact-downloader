# harvester-artifact-downloader
This script will download the artifacts for seeder and the pxe boot for both amd64 and arm64.
```
Syntax: iso [version] [version-output] -s
options:
[version]            version to be downloaded.
[version-output]     This is an optional output if you need a different name.
-s                   This is optional if you need to symlink the destination directory to a existing release.
```
